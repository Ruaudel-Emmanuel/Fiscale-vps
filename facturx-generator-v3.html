How V3 works

  • Frontend: collects invoice data and exports a clean JSON payload.
  • Backend Python: generates a readable invoice PDF, builds the invoice XML, then embeds the XML into a PDF/A-3 container.
  • Validation: the resulting file should be checked with veraPDF and Factur-X/EN16931 validators before production use.
  • Important: browser JavaScript alone is not the right tool for guaranteed PDF/A-3 + XMP + embedded XML compliance.
